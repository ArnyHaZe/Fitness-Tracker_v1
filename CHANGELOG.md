# Changelog

All notable changes to Fitness Tracker v1 will be documented in this file.

## v1.0.0

Initial release.

### Added

- 26-week strength and recomposition tracker
- Weekly cut / bulk phase toggle
- Workout logging for weight, reps, and RPE
- Suggested load progression
- Exercise swaps
- Rest timer
- Weekly check-ins
- Body-weight progress chart
- Strength progress chart
- Local-first saving with browser localStorage
- Web app manifest
- App icons
- Service worker for offline caching
- GitHub Pages deployment support
